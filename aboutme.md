I’m a PhD student in structural biology of RNA binding proteins. I'm working on using FRET to study conformational changes of a family of designer RNA binding proteins known as PPRs. More details to come when I finally get my first paper published.

My current passion is finding new ways to visualise biological data. I've managed to pick up some skills using Blender for 3D visualisation. Slowly picking up coding skills in `R` and `Python` to hopefully wrangle large datasets better.

These sorts of data wrangling and visualisation tools weren't taught at any level, so I hope to help fill this knowledge gap. So far I have helped run an intro to [R for biologists workshop](https://hackmd.io/6mexDqmdRfC7zEk9ESct7w) and have started a [YouTube channel](https://youtube.com/bradyjohnston) to teach visualisation of proteins using the free, open-source and very powerful program [Blender](https://blender.org).
